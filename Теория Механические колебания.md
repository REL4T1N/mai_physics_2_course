# 1. Свободные незатухающие гармонические колебания и их характеристики. Математический, пружинный и физический маятники.

## Свободные незатухающие гармонические колебания и их характеристики
**Колебание** - это периодическое движение около положения равновесия, подчиняющееся закону
$$x(t)=x(t+T)$$
**Период $T$** - время, одного полного колебания
**Частота $\nu$** - количество колебаний за единицу времени: $\nu = \frac{1}{T}$
**Круговая частота**: $\omega = 2\pi\nu = \frac{2\pi}{T}$

**Гармоническими колебаниями** называются периодические колебания, подчиняющиеся **закону синуса**:
$$\large x(t)=Asin(\omega_0 t + \alpha),$$
где $A$ - амплитуда, $\omega_0$ - круговая частота колебаний, $\omega_0 t + \alpha$ - фаза, $\alpha$ - фаза при $t=0$.
Система совершающая гармонические колебания, называется **колебательной системой**.

Скорость при гармонических колебаниях: $v = \frac{dx}{dt} = A\omega_0 cos(\omega_0 t + \alpha)$
Ускорение при гармонических колебаниях: $a = \frac{d^2 x}{dt} = -A\omega_0^2 sin(\omega_0 t + \alpha)$
Также ускорение можно записать как: $\large a = -\omega_0^2 x$

**Дифференциальное уравнение гармонических колебаний:**
$$\large \ddot{x} + \omega_0^2 x = 0$$
Из формулы ускорения через координаты и второго закона Ньютона получаем:
$$F = -m \omega_0^2x, \quad k = m\omega_0^2 \quad => \quad \vec{F} = -k \vec{x}$$
Это называется **возвращающей силой**, а $k$ называется **коэффициентом возвращающей силы**

Колебания, происходящие под действием сил, называются *собственными*
Круговая частота ГК: $\omega_0 = \sqrt{\frac{k}{m}}$
Период ГК: $T = 2\pi \sqrt{\frac{m}{k}}$
Кинетические энергия при ГК: $\large K = \frac{m v^2}{2}= \frac{m A^2 \omega_0^2 cos^2(\omega_0 t + \alpha)}{2}$
Потенциальная энергия при ГК: $\large U = \frac{kx^2}{2} = \frac{mA^2 \omega_0^2 sin^2(\omega_0 t + \alpha)}{2}$

## Пружинный маятник
Представляет собой груз подвязанный на пружине. В данном случае $F = -kx$ является силой упругости, а $k$ - коэффициентом упругости. При отсутствии сил сопротивления колебания подчиняются стандартному закону синуса:
$$\large x(t)=Asin(\omega_0 t + \alpha)$$

## Физический маятник
**Физический маятник** - твёрдое тело, совершающее колебания вокруг закреплённой оси, не совпадающей с центром масс. Колебания являются гармоническими **при малых отклонениях.**
**Момент силы** ФМ: $\vec{M} = mgdsin\varphi \approx mgd\vec{\varphi}$, $d$ - расстояние от начала выбранной оси до тела
**Основное уравнение динамики вращательного движения** ФМ: $M = I\varepsilon,$ где $I$ момент инерции оси, $\varepsilon$ - угловое ускорение
Круговая частота ФМ: $\omega_0 = \sqrt{\frac{mgd}{I}}$
Период ФМ: $T = 2\pi \sqrt{\frac{I}{mgd}}$

## Математический маятник
**Математический маятник** является частным случаем физического маятника и представляет собой **материальную точку**, подвешенную на *невесомой нерастяжимой нити* длиной $L$. Тогда $d = L, \quad I = mL^2$.
Круговая частота ММ: $\omega_0 = \sqrt{\frac{g}{L}}$
Период ММ: $T = 2\pi \sqrt{\frac{L}{g}}$

# 2. Свободные затухающие колебания и их характеристики. Вынужденные колебания. Явление резонанса

## Свободные затухающие колебания и их характеристики
**Свободные колебания** называются **затухающими** если присутствует **сила сопротивления**: $\vec{F}_{\text{сопр}} = -r\vec{v} = -r\dot{x}$

По второму закону Ньютона получаем: 
$$m\ddot{x} = -kx - r\dot{x}$$
Тогда **дифференциальное уравнение затухающий колебаний имеет вид**:
$$\large \ddot{x} + 2\beta \dot{x} + \omega_0^2 x = 0,$$
где $\beta = \frac{r}{2m}$ - коэффициент затухания, а $\omega_0 = \sqrt{\frac{k}{m}}$

Если $\omega_0^2 > \beta^2$, то решением **ДУЗК** является $\large x(t) = A_0 e^{-\beta t} sin(\omega t + \alpha),$ где $\omega = \sqrt{\omega_0^2 - \beta^2}$ - угловая частота затухающий колебания, $A_0 e^{-\beta t}$ - амплитуда затухающих колебаний.
Период затухающих колебаний: $T = \frac{2 \pi}{\omega} = \frac{2 \pi}{\sqrt{\omega_0^2 - \beta^2}}$

Логарифмический декремент затухания $\delta$ - постоянная величина, характеризующая скорость затухания. Вычисляется по формуле
$$\delta = \ln\frac{A(t)}{A(t+T)} = \beta T$$

## Вынужденные колебания
Если на колеблющуюся точку, кроме сил упругости и сопротивления, действует внешняя сила, изменяющаяся по гармоническому закону, то колебания называются **вынужденными**.

**Дифференциальное уравнение вынужденных колебаний:**
$$\large \ddot{x} + 2\beta \dot{x} + \omega_0^2 x = \frac{F_0 sin (\Omega t)}{m}$$
Решением **ДУВК** ,будет $\large x(t) = A_0 e^{-\beta t} sin(\omega t + \alpha) + A sin(\Omega t + \psi)$
После затухания происходят вынужденные колебания с частотой $\Omega$ по закону $\large x(t) = sin(\Omega t + \psi)$. Амплитуда будет $\large A = \frac{F_0}{m \sqrt{(\omega_0^2 - \Omega^2)^2 + 4\beta^2 \Omega^2}}$
Сдвиг фаз между колебаниями точки и вынуждающей силы: $tg\psi = - \frac{2\beta \Omega}{\omega_0^2 - \Omega^2}$

## Явление резонанса
**Резонанс** - резкое возрастание амплитуды колебаний при определённой частоте $\omega_{\text{рез}}$ вынуждающей силы. $\omega_{\text{рез}} = \sqrt{\omega_0^2 - 2\beta^2}$
Амплитуда колебаний при резонансе: $\large A_{\text{рез}} = \frac{F_0}{2m \beta \sqrt{\omega_0^2 - \beta^2}}$
Чем меньше коэффициент затухания $\beta$, тем круче резонансная кривая
