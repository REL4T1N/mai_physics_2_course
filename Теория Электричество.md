# 1. Элементарный заряд. Закон сохранения заряда. Закон Кулона. Напряжённость как силовая характеристика электрического поля. Принцип суперпозиции. Силовые линии электростатического поля.

## Элементарный заряд. Закон сохранения заряда
**Элементарный заряд** - физический параметр тела, определяющий его взаимодействие с электромагнитным полем
**Свойства**:
- положительный и отрицательный
- $e$ - электрон - мельчайший электрический заряд, равен $1.6 * 10^9$ Кл.
- Любой заряд равен сумме какое-то количества $e$: $q = eN$
- В изолированной системе сумма зарядов постоянна

## Закон Кулона
$$\large {F = \frac{k q_1 q_2}{\varepsilon r^2}},$$
где $k = \frac{1}{4 \pi \varepsilon_0}$, $\varepsilon_0$ - электрическая постоянная, $\varepsilon$ - диэлектрическая проницаемость.

## Напряжённость как силовая характеристика электрического поля
**Напряженностью электростатического поля $\vec{E}$** - векторная величина, вычисляемая по формуле: 
$$\vec{E} = \frac{\vec{F}}{q}$$
**Электрическое поле называется однородным**, если его напряжённость по всех точках одинакова.
**Напряжённость точечного заряда**: $\large {E = \frac{kq}{\varepsilon r^2}}$

## Принцип суперпозиции.
Принцип **суперпозиции (сложения)** электрических полей:
$$\vec{E} = \sum_i \vec{E}_i$$

## Силовые линии электростатического поля.
Силовой линией вектора напряженности $\vec{E}$ называется линия, касательная в каждой точке к которой, совпадает по направлению с вектором напряженности $\vec{E}$ в этой точке.

## 2. Поток вектора напряжённости. Теорема Остроградского – Гаусса для вектора напряжённости электростатического поля. Примеры применения теоремы.

## Поток вектора напряжённости
**Потоком вектора напряженности** электростатического поля через поверхность S называется величина, равная
$$\large \Phi = \int_S E_n dS,$$
где $E_n$ - проекция $\vec{E}$ на **вектор нормали $\vec{n}$** к элементу поверхности ($E_n = Ecos(\alpha)$)

## Теорема Остроградского – Гаусса для вектора напряжённости электростатического поля
Поток вектора напряженности электростатического поля через произвольную замкнутую поверхность равен алгебраической сумме зарядов, заключенных внутри этой поверхности, делённой на электрическую постоянную:
$$\large \Phi = \oint_S E_n dS = \frac{\sum_i q_i}{\varepsilon_0}$$
Теорема **Остроградского-Гаусса** используется для для расчёта **напряжённости электростатического поля** заряженных тел. Важное значение имеет выбор *вспомогательной замкнутой поверхности*

# 3. Работа сил электростатического поля. Потенциал как энергетическая характеристика электростатического поля. Циркуляция вектора напряжённости. Связь между напряжённостью и потенциалом. Эквипотенциальные поверхности. 

## Работа сил электростатического поля
Работа сил по перемещению единичного положительного заряда в поле напряженностью $\vec{E}$ их точки 1 в точку 2:
$$A = \int_1^2 dA = \int_1^2 \vec{F} d\vec{r}$$

## Потенциал как энергетическая характеристика электростатического поля
Энергетической характеристикой поля называется **потенциал $\varphi$** - скалярная характеристика электростатического поля:
$$\large \varphi = \frac{U}{q}$$
**Потенциал поля** произвольной точки определяется работой поля при перемещении единичного положительного заряда из данной точки поля в бесконечность
$$\large \varphi =  \frac{A}{q}=\int\limits_r^\infty \vec{F} d\vec{r}$$
**Потенциал поля точечного заряда** на расстоянии $r$: $\large{\varphi = \frac{kq}{r}}$

## Циркуляция вектора напряжённости
Работа сил по перемещению единичного положительного заряда в поле напряженностью $\vec{E}$ их точки 1 в точку 2:
$$A = \int_1^2 dA = \int_1^2 \vec{E} d\vec{L}$$
Если точки 1 и 2 совпадают, то интеграл по замкнутому контору $L$ называется циркуляцией вектора $\vec{E}$
**Теорема о циркуляции вектора напряженности электрического поля $\vec{E}$**: 
$$\large \oint_l \vec{E}d\vec{l} = 0 $$**Словесно:** Циркуляция вектора напряженности потенциального электрического поля по замкнутому контору равна нулю.

## Связь между напряжённостью и потенциалом. 
**Потенциал** и **напряженность электрического поля** связаны формулой: 
$$\large \vec{E} = -grad(\varphi),$$знак "-" означает, что они **противоположно** направлены.
В трёхмерном пространстве:
$$\vec{E} = \vec{E_x} + \vec{E_y} + \vec{E_z} = -\left(\frac{\partial \varphi}{\partial x}\vec{i} + \frac{\partial \varphi}{\partial y}\vec{j} + \frac{\partial \varphi}{\partial z}\vec{k} \right)$$

## Эквипотенциальные поверхности
**Эквипотенциальная поверхность** - поверхность, все точки которой имеют одинаковый потенциал. Эквипотенциальные поверхности одного поля не пересекаются между собой.

**Уравнение** эквипотенциальной поверхности имеет вид $\varphi = const$, для точечного заряда получается $r = const$ или $x^2 + y^2 + z^2 = const$. Получается, что для точечных зарядов эквипотенциальные поверхности это сферы, центры которых совпадают с зарядом.

Работа по перемещению заряда между точками *1* и *2* эквипотенциальной поверхности равна нулю: $A_{12} = q(\varphi_{r_1} - \varphi_{r_2}) = 0$.
