
# Вращательное движение твёрдого тела

## **I. ОСНОВНАЯ ТЕОРИЯ**

### **1. Кинематика вращения**
*   **Вращательное движение** — движение, при котором все точки тела движутся по окружностям, центры которых лежат на **неподвижной оси**.
*   **Угол поворота** ($\varphi$) — скалярная мера ориентации тела [рад].
*   **Вектор элементарного поворота** ($d\vec{q}$) направлен по оси вращения по **правилу правого винта**.
*   **Угловая скорость** ($\vec{\omega}$) — векторная величина, характеризующая быстроту и направление вращения.
    $$\vec{\omega} = \frac{d\vec{q}}{dt} = \frac{d\varphi}{dt}$$
*   **Угловое ускорение** ($\vec{\epsilon}$) — характеризует изменение угловой скорости.
    $$\vec{\epsilon} = \frac{d\vec{\omega}}{dt}$$
    *Направление:* совпадает с $\vec{\omega}$ при ускорении, противоположно при замедлении.
*   **Связь линейных и угловых величин** для точки на расстоянии $R$ от оси:
    $$\large \vec{v} = [\vec{\omega}, \vec{R}], \quad v = \omega R$$
    $$\large \vec{a} = \vec{a}_n + \vec{a}_\tau = -\omega^2 \vec{R} + [\vec{\epsilon}, \vec{R}], \quad a_n = \omega^2 R, \ a_\tau = \epsilon R$$

### **2. Динамика: Моменты**
*   **Момент силы** относительно точки $O$: $\vec{M}_O = [\vec{r}, \vec{F}]$
    *   *Модуль:* $M = F \cdot L$, где $L$ — **плечо силы**.
    *   *Физический смысл:* Мера вращательного действия силы.
*   **Момент импульса** (момент количества движения):
    *   Для точки: $\vec{L}_O = [\vec{r}, m\vec{v}]$
    *   **Физический смысл:** Мера количества вращательного движения.
*   **Уравнение моментов** (основной закон динамики для точки):
    $$\large \frac{d\vec{L}_O}{dt} = \vec{M}_O$$

### **3. Момент инерции и основное уравнение**
*   **Момент инерции** ($J$) — **мера инертности** во вращательном движении (аналог массы).
    *   Для системы точек: $J = \sum_i m_i R_i^2$
    *   **Свойство:** Зависит от распределения массы относительно оси.
    *   *Единица:* $кг \cdot м^2$.
*   **Момент импульса тела** относительно оси:
    $$\large L_z = J \omega$$
*   **Основное уравнение динамики вращательного движения твёрдого тела:**
    $$\large M_z = J \epsilon$$
    *Аналогия со вторым законом Ньютона:* $F = ma$.

### **4. Теорема Штейнера**
*   **Формулировка:** Момент инерции тела относительно произвольной оси равен сумме момента инерции относительно оси, проходящей через **центр масс** параллельно данной, и произведения массы тела на квадрат расстояния между осями.
    $$\large J = J_C + m a^2$$
*   **Практическое значение:** Позволяет вычислять $J$ для сложных тел и осей.

### **5. Энергетика и законы сохранения**
*   **Кинетическая энергия вращения:**
    $$\large K_{\text{вр}} = \frac{J \omega^2}{2}$$
*   **Работа при вращении:** $\delta A = M_z d\varphi$
*   **Закон сохранения момента импульса:**
    Если суммарный момент внешних сил относительно оси равен нулю ($\sum M_z^{\text{внеш}} = 0$), то
    $$\large L_z = \sum J_i \omega_i = \text{const}$$
    *Следствие*: Увеличение момента инерции системы приводит к уменьшению угловой скорости, и наоборот (эффект фигуриста).

### **6. Сводная таблица аналогий**
| **Поступательное движение** | **Вращательное движение** |
| :--- | :--- |
| Путь $s$ | Угол $\varphi$ |
| Масса $m$ | Момент инерции $J$ |
| Сила $\vec{F}$ | Момент силы $\vec{M}$ |
| Импульс $\vec{p}=m\vec{v}$ | Момент импульса $\vec{L}=J\vec{\omega}$ |
| Основной закон: $\vec{F}=m\vec{a}$ | Основной закон: $\vec{M}=J\vec{\epsilon}$ |
| Кинетическая энергия: $\frac{mv^2}{2}$ | Кинетическая энергия: $\frac{J\omega^2}{2}$ |
| ЗСИ: $\sum \vec{F}=0 \Rightarrow \vec{p}=const$ | ЗСМИ: $\sum \vec{M}=0 \Rightarrow \vec{L}=const$ |

## **II. ПРАКТИЧЕСКИЕ ПРИЛОЖЕНИЯ (Лабораторные работы)**

### **ЛР 7(ф). Расчёт момента инерции сложных тел**
*   **Цель:** Освоить методы теоретического расчёта $J$ для тел сложной формы.
*   **Ключевые методы:**
    1.  **Интегрирование:** Для тел правильной формы (диск: $J = \frac{mR^2}{2}$).
    2.  **Метод сложения/вычитания:** Тело представляется как комбинация простых тел.
        *Пример:* Диск с отверстием: $J_{\text{общ}} = J_{\text{диск}} - J_{\text{отверстие}}$.
    3.  **Комбинация с [[#4. Теорема Штейнера | теоремой Штейнера]]:** Если оси простых тел не совпадают с общей осью.
*   **Итоговая формула для диска со смещённым отверстием:**
    $$\large J_0 = \frac{m}{8} \left[ \frac{D^2}{1 - (d/D)^2} - \frac{d^2 + 2(2L + d - D)^2}{(D/d)^2 - 1} \right]$$

### **ЛР 1. Маятник Максвелла**
*   **Цель:** Экспериментально определить $J$, используя **[[#5. Энергетика и законы сохранения | закон сохранения энергии]]**.
*   **Суть:** Тело, подвешенное на бифилярном подвесе, движется, превращая потенциальную энергию в кинетическую энергию **поступательного** и **вращательного** движения.
*   **Связь скоростей:** $v = \omega r$ (нить не проскальзывает).
*   **Ключевой вывод:** Ускорение центра масс постоянно и меньше $g$:
    $$\large a = \frac{g}{1 + \frac{J}{mr^2}}$$
*   **Рабочая формула:**
    $$\large J_{\text{эксп}} = mr^2 \left( \frac{g}{a} - 1 \right), \quad \text{где } a = 2k^2 \text{ (определяется из графика } \sqrt{S}(t) \text{)}$$
*   **Связь с теорией:** Прямое применение [[#5. Энергетика и законы сохранения | ЗСЭ]] для системы с двумя видами движения.

### **ЛР 3. Маятник Обербека (Крестовина)**
*   **Цель:** Экспериментальная проверка **[[#3. Момент инерции и основное уравнение | основного уравнения динамики вращательного движения]]** $M = J\epsilon$.
*   **Суть:** Падающий груз создаёт момент силы, раскручивающий крестовину с перемещаемыми грузами.
*   **Система уравнений:**
    1.  Для груза: $mg - T = ma$
    2.  Для крестовины: $M = Tr = J\epsilon$
    3.  Кинематическая связь: $\epsilon = a/r$
*   **Рабочая формула для $J_{\text{эксп}}$:**
    $$\large J_{\text{эксп}} = mr^2 \left( \frac{g}{a} - 1 \right), \quad a = \frac{2h}{t^2}$$
*   **Теоретический расчёт $J_{\text{теор}}$ (применение [[#4. Теорема Штейнера | теоремы Штейнера]] и аддитивности):**
    $$\large J_{\text{теор}} = J_0 + 4m_1R^2 + \frac{4}{3}m_2L^2$$

### **ЛР 6. Баллистический маятник**
*   **Цель:** Определить скорость снаряда, используя **[[#5. Энергетика и законы сохранения | законы сохранения момента импульса и энергии]]**.
*   **Физическая модель:**
    *   **Этап 1 (удар):** **Абсолютно неупругий удар**. Применяется **ЗСМИ**, так как момент внешних сил равен нулю.
        $$\large m_n v L = J \omega_0$$
    *   **Этап 2 (отклонение):** Применяется **ЗСЭ** для крутильных колебаний.
        $$\large \frac{J\omega_0^2}{2} = \frac{c \varphi^2}{2}$$
*   **Итоговая формула для скорости:**
    $$\large v = \frac{\varphi \sqrt{J c}}{m_n L}$$
*   **Расчёт момента инерции маятника $J_M$** (метод двух опытов для исключения констант):
    $$\large J_M = \frac{2m(a_2^2 \varphi_2^2 - a_1^2 \varphi_1^2)}{\varphi_1^2 - \varphi_2^2} - 2m \left( \frac{r^2}{4} + \frac{h^2}{12} \right)$$
*   **Связь с теорией:** Классический пример разделения быстрого (удар, ЗСМИ) и медленного (колебания, ЗСЭ) процессов.

---

1. **Кинематические характеристики вращательного движения тела.**  
	Угол поворота $\varphi$ (рад), угловая скорость $\vec{\omega} = d\varphi/dt$ (рад/с), угловое ускорение $\vec{\epsilon} = d\vec{\omega}/dt$ (рад/с²). Все векторы направлены вдоль оси вращения по правилу правого винта.

2. **Нормальное и тангенциальное ускорение тела.**  
	Для точки на расстоянии $R$ от оси:  
	Тангенциальное ускорение $a_\tau = \epsilon R$ — изменяет модуль скорости.  
	Нормальное (центростремительное) ускорение $a_n = \omega^2 R$ — изменяет направление скорости.

3. **Понятие момента силы относительно неподвижной точки. Каковы единицы измерения момента силы?**  
	Момент силы $\vec{M} = [\vec{r}, \vec{F}]$ — вектор, равный векторному произведению радиуса-вектора точки приложения силы на силу.  
	Единица измерения: Н·м.

4. **Момент импульса относительно неподвижной точки. Уравнение моментов.**  
	Момент импульса материальной точки: $\vec{L} = [\vec{r}, m\vec{v}]$.  
	Уравнение моментов: $\frac{d\vec{L}}{dt} = \vec{M}$ — производная момента импульса равна моменту силы.

5. **Закон сохранения момента импульса для системы материальных точек.**  
	Если сумма моментов внешних сил относительно неподвижной точки равна нулю, то момент импульса системы постоянен: $\vec{L} = \sum \vec{L}_i = const$.

6. **Понятие момента силы относительно оси.**  
	Это проекция вектора момента силы относительно точки на оси на данную ось. Определяется только тангенциальной составляющей силы.

7. **Понятие момента импульса твердого тела относительно оси.**  
	Момент импульса тела, вращающегося вокруг оси: $L_z = J \omega$, где $J$ — момент инерции, $\omega$ — угловая скорость.

8. **Написать основное уравнение динамики вращательного движения.**  
	$M_z = J \epsilon$, где $M_z$ — момент сил относительно оси, $J$ — момент инерции, $\epsilon$ — угловое ускорение.

9. **Что такое момент инерции тела? Каков его физический смысл?**  
	Момент инерции $J = \sum m_i R_i^2$ — скалярная мера инертности тела во вращательном движении. Физический смысл: аналог массы для вращения; чем больше $J$, тем труднее раскрутить или остановить тело.

10. **Расчет момента инерции стержня относительно оси, проходящей через центр масс перпендикулярно стержню.**  
	$J = \frac{m L^2}{12}$, где $m$ — масса, $L$ — длина стержня.

11. **Расчет момента инерции диска относительно оси, проходящей через центр масс перпендикулярно плоскости диска.**  
	$J = \frac{m R^2}{2}$, где $m$ — масса, $R$ — радиус диска.

12. **Теорема Штейнера, пример ее применения.**  
	$J = J_C + m a^2$, где $J_C$ — момент инерции относительно оси через центр масс, $a$ — расстояние между осями.  
	Пример: расчёт момента инерции стержня относительно оси, проходящей через его конец: $J = \frac{mL^2}{12} + m(\frac{L}{2})^2 = \frac{mL^2}{3}$.

13. **Кинетическая энергия вращающегося тела.**  
	$K = \frac{J \omega^2}{2}$. Аналог кинетической энергии поступательного движения $K = \frac{m v^2}{2}$.

14. **Закон сохранения момента импульса для твердого тела, вращающегося относительно неподвижной оси.**  
	Если сумма моментов внешних сил относительно оси равна нулю, то момент импульса сохраняется: $J\omega = const$. Следствие: при уменьшении момента инерции угловая скорость увеличивается.

## **Ответы на вопросы раздела (подробно):**
**1. Кинематические характеристики вращательного движения тела.**  
Основными кинематическими характеристиками вращательного движения твёрдого тела вокруг неподвижной оси являются:  
*   **Угол поворота** $\varphi$ — скалярная величина, измеряемая в радианах (рад), характеризующая изменение ориентации тела.  
*   **Угловая скорость** $\vec{\omega}$ — векторная величина, характеризующая быстроту и направление вращения. Она определяется как производная вектора элементарного поворота по времени:
    $$\vec{\omega} = \frac{d\vec{q}}{dt}, \quad \omega = \frac{d\varphi}{dt}$$
    Единица измерения — радиан в секунду (рад/с). Вектор $\vec{\omega}$ направлен вдоль оси вращения по правилу правого винта.  
*   **Угловое ускорение** $\vec{\epsilon}$ — векторная величина, характеризующая быстроту изменения угловой скорости:
    $$\vec{\epsilon} = \frac{d\vec{\omega}}{dt}$$
    Единица измерения — радиан на секунду в квадрате (рад/с²). Вектор $\vec{\epsilon}$ совпадает по направлению с $\vec{\omega}$ при ускоренном вращении и противоположен ему при замедленном.

**2. Нормальное и тангенциальное ускорение тела.**  
При вращении твёрдого тела ускорение любой его точки складывается из двух взаимно перпендикулярных составляющих:  
*   **Тангенциальное (касательное) ускорение** $a_\tau$ — характеризует изменение модуля линейной скорости точки. Оно направлено по касательной к траектории точки (окружности) и связано с угловым ускорением:  
    $$a_\tau = \epsilon R$$
    где $R$ — расстояние от точки до оси вращения.  
*   **Нормальное (центростремительное) ускорение** $a_n$ — характеризует изменение направления линейной скорости. Оно направлено к центру окружности (к оси вращения) и связано с угловой скоростью:  
    $$a_n = \omega^2 R$$
    Полное ускорение точки: $\vec{a} = \vec{a}_\tau + \vec{a}_n$, его модуль $a = \sqrt{a_\tau^2 + a_n^2}$.

**3. Понятие момента силы относительно неподвижной точки. Каковы единицы измерения момента силы?**  
**Моментом силы $\vec{M}$** относительно неподвижной точки $O$ называется векторная величина, равная векторному произведению радиуса-вектора $\vec{r}$, проведённого из точки $O$ в точку приложения силы, на вектор силы $\vec{F}$:  
$$\vec{M} = [\vec{r}, \vec{F}]$$  
*   **Модуль момента силы:** $M = rF \sin\alpha = F \cdot L$, где $\alpha$ — угол между векторами $\vec{r}$ и $\vec{F}$, а $L = r \sin\alpha$ — **плечо силы** (кратчайшее расстояние от точки $O$ до линии действия силы).  
*   **Направление** вектора $\vec{M}$ определяется правилом правого винта (или правилом буравчика): он перпендикулярен плоскости, в которой лежат векторы $\vec{r}$ и $\vec{F}$.  
*   **Единица измерения** в СИ: ньютон-метр (Н·м).

**4. Момент импульса относительно неподвижной точки. Уравнение моментов.**  
**Моментом импульса (моментом количества движения) $\vec{L}$** материальной точки относительно неподвижной точки $O$ называется вектор, равный векторному произведению радиуса-вектора $\vec{r}$ этой точки на вектор её импульса $m\vec{v}$:  
$$\vec{L} = [\vec{r}, m\vec{v}]$$  
**Уравнение моментов** (основной закон динамики вращательного движения точки) устанавливает связь между моментом импульса и моментом силы:  
$$\frac{d\vec{L}}{dt} = \vec{M}$$  
*Скорость изменения момента импульса материальной точки равна моменту приложенной к ней силы.*  
Это — вращательный аналог второго закона Ньютона $\frac{d\vec{p}}{dt} = \vec{F}$.

**5. Закон сохранения момента импульса для системы материальных точек.**  
Для системы материальных точек закон сохранения момента импульса формулируется следующим образом:  
**Если векторная сумма моментов всех внешних сил, действующих на систему, относительно некоторой неподвижной точки равна нулю, то момент импульса системы относительно этой точки остаётся постоянным.**  
Математически: если $\sum \vec{M}_{внеш} = 0$, то  
$$\vec{L}_{системы} = \sum_{i} \vec{L}_i = const$$  
Этот закон является фундаментальным и выполняется для замкнутых систем, а также в случаях, когда результирующий момент внешних сил равен нулю.

**6. Понятие момента силы относительно оси.**  
**Моментом силы относительно оси** называется скалярная величина, равная проекции на эту ось вектора момента силы, вычисленного относительно любой точки, лежащей на оси.  
*   **Физический смысл:** Момент силы относительно оси характеризует вращательное действие силы именно вокруг этой оси.  
*   **Важное свойство:** Вращение вокруг оси вызывается только **тангенциальной** составляющей силы. Осевая и радиальная составляющие момента относительно этой оси равны нулю.

**7. Понятие момента импульса твердого тела относительно оси.**  
Для твёрдого тела, вращающегося вокруг неподвижной оси, **моментом импульса относительно этой оси** называется проекция вектора момента импульса тела на ось вращения. Он выражается через момент инерции тела и его угловую скорость:  
$$L_z = J \omega$$  
где $J$ — момент инерции тела относительно оси вращения, $\omega$ — угловая скорость.  
*   **Вывод:** $L_z = \sum_i m_i v_i R_i = \sum_i m_i (\omega R_i) R_i = \omega \sum_i m_i R_i^2 = J\omega$.

**8. Написать основное уравнение динамики вращательного движения.**  
Основное уравнение динамики вращательного движения твёрдого тела вокруг неподвижной оси имеет вид:  
$$M_z = J \epsilon$$  
где:  
*   $M_z$ — результирующий момент всех внешних сил относительно оси вращения,  
*   $J$ — момент инерции тела относительно той же оси,  
*   $\epsilon = \frac{d\omega}{dt}$ — угловое ускорение тела.  
Это уравнение является вращательным аналогом второго закона Ньютона $F = ma$.

**9. Что такое момент инерции тела? Каков его физический смысл?**  
**Момент инерции $J$** — это скалярная физическая величина, являющаяся мерой инертности тела во вращательном движении вокруг оси.  
*   **Определение:** $J = \sum_i m_i R_i^2$, где $m_i$ — масса i-й материальной точки, $R_i$ — её расстояние до оси вращения. Для сплошного тела $J = \int_V R^2 dm$.  
*   **Физический смысл:** Момент инерции играет ту же роль во вращательном движении, что масса — в поступательном. Чем больше момент инерции тела, тем труднее его раскрутить или остановить (тем меньше угловое ускорение при том же моменте сил). Он зависит не только от массы тела, но и от её распределения относительно оси вращения: чем дальше масса распределена от оси, тем больше момент инерции.  
*   **Единица измерения:** $кг \cdot м^2$.

**10. Расчет момента инерции стержня относительно оси, проходящей через центр масс перпендикулярно стержню.**  
Рассмотрим тонкий однородный стержень массы $m$ и длины $L$. Ось вращения $Z$ проходит через его центр масс перпендикулярно стержню.  
Линейная плотность стержня: $\lambda = m/L$.  
Выделим малый элемент длиной $dx$ на расстоянии $x$ от центра. Его масса $dm = \lambda dx$. Момент инерции этого элемента: $dJ = x^2 dm = \lambda x^2 dx$.  
Интегрируя по длине стержня от $-L/2$ до $L/2$, получаем:  
$$J = \int_{-L/2}^{L/2} \lambda x^2 dx = \lambda \frac{x^3}{3} \Big|_{-L/2}^{L/2} = \frac{\lambda}{3} \left( \frac{L^3}{8} + \frac{L^3}{8} \right) = \frac{\lambda L^3}{12}$$  
Подставляя $\lambda = m/L$, получаем окончательно:  
$$J_C = \frac{mL^2}{12}$$

**11. Расчет момента инерции диска относительно оси, проходящей через центр масс перпендикулярно плоскости диска.**  
Рассмотрим тонкий однородный диск (или цилиндр) массы $m$ и радиуса $R$. Ось вращения $Z$ проходит через его центр перпендикулярно плоскости диска.  
Поверхностная плотность диска: $\sigma = \frac{m}{\pi R^2}$.  
Разобьём диск на тонкие концентрические кольца радиусом $r$ и шириной $dr$. Площадь такого кольца $dS = 2\pi r dr$, его масса $dm = \sigma dS = \sigma \cdot 2\pi r dr$. Все точки кольца находятся на одинаковом расстоянии $r$ от оси, поэтому момент инерции кольца: $dJ = r^2 dm = 2\pi\sigma r^3 dr$.  
Интегрируя по радиусу от $0$ до $R$, получаем:  
$$J = \int_{0}^{R} 2\pi\sigma r^3 dr = 2\pi\sigma \frac{R^4}{4} = \frac{\pi\sigma R^4}{2}$$  
Подставляя $\sigma = \frac{m}{\pi R^2}$, получаем окончательно:  
$$J_C = \frac{mR^2}{2}$$

**12. Теорема Штейнера, пример ее применения.**  
**Теорема Штейнера (о параллельных осях):** Момент инерции тела $J$ относительно произвольной оси равен сумме момента инерции этого тела $J_C$ относительно оси, проходящей через его центр масс параллельно данной, и произведения массы тела $m$ на квадрат расстояния $a$ между осями:  
$$J = J_C + m a^2$$  
**Пример применения:** Вычислим момент инерции того же однородного стержня (масса $m$, длина $L$) относительно оси, проходящей через его конец и перпендикулярной стержню.  
Расстояние между осями $a = L/2$. Момент инерции относительно центра $J_C = \frac{mL^2}{12}$.  
По теореме Штейнера:  
$$J_{конец} = J_C + m\left(\frac{L}{2}\right)^2 = \frac{mL^2}{12} + \frac{mL^2}{4} = \frac{mL^2}{3}$$

**13. Кинетическая энергия вращающегося тела.**  
Кинетическая энергия твёрдого тела, вращающегося вокруг неподвижной оси со скоростью $\omega,$ равна сумме кинетических энергий всех его материальных точек:  
$$K = \sum_i \frac{m_i v_i^2}{2} = \sum_i \frac{m_i (\omega R_i)^2}{2} = \frac{\omega^2}{2} \sum_i m_i R_i^2$$  
Учитывая, что $\sum_i m_i R_i^2 = J$, получаем формулу для кинетической энергии вращения:  
$$K = \frac{J \omega^2}{2}$$  
Эта формула является полным аналогом формулы кинетической энергии поступательного движения $K = \frac{m v^2}{2}$, где масса $m$ заменена на момент инерции $J$, а скорость $v$ — на угловую скорость $\omega$.

**14. Закон сохранения момента импульса для твердого тела, вращающегося относительно неподвижной оси.**  
Для твёрдого тела или системы тел, вращающихся вокруг неподвижной оси, закон сохранения момента импульса формулируется следующим образом:  
**Если сумма моментов всех внешних сил относительно неподвижной оси вращения равна нулю, то момент импульса системы относительно этой оси сохраняется.**  
Математически: если $M_z^{сум} = 0$, то  
$$L_z = J\omega = const$$  
где $J$ — момент инерции системы, $\omega$ — её угловая скорость.  
*   **Важное следствие:** Поскольку $J\omega = const$, то изменение момента инерции системы (например, за счёт перераспределения масс) приводит к обратно пропорциональному изменению её угловой скорости: $J_1\omega_1 = J_2\omega_2$. Это явление используется фигуристами для изменения скорости вращения.
