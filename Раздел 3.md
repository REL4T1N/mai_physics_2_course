# Механические колебания и волны  

---
## **1. Основные понятия колебаний**
### 1.1 Определения
- **Колебания** – повторяющиеся во времени процессы.
- **Периодические колебания** – колебания, при которых значения физических величин повторяются через равные промежутки времени.
- **Период $T$** – минимальное время одного полного колебания.
- **Частота $f$** – число колебаний в единицу времени:  $$f = \frac{1}{T}$$
- **Круговая (циклическая) частота $\omega$**:  $$\omega = 2\pi f$$
  Связь с периодом:  $$T = \frac{2\pi}{\omega}$$
- Условие периодичности: $x(t) = x(t + T)$ для любого $t$.

---

## **2. Гармонические колебания**
### 2.1 Уравнение гармонических колебаний
$$x(t) = A \sin(\omega_0 t + \alpha)$$
- $A$ – **амплитуда** (максимальное смещение)
- $\omega_0$ – круговая частота собственных колебаний
- $\alpha$ – начальная фаза
- Фаза колебаний: $(\omega_0 t + \alpha)$

### 2.2 Скорость и ускорение
- **Скорость**:
 $$v = \frac{dx}{dt} = A\omega_0 \cos(\omega_0 t + \alpha)$$
- **Ускорение**:
  $$a = \frac{d^2x}{dt^2} = -A\omega_0^2 \sin(\omega_0 t + \alpha)$$

### 2.3 Важное соотношение
$$a = -\omega_0^2 x$$
Ускорение пропорционально смещению и направлено противоположно.

---

## **3. Дифференциальное уравнение гармонических колебаний**
$$\ddot{x} + \omega_0^2 x = 0$$
**Решение**: $x = A \sin(\omega_0 t + \alpha)$

### 3.1 Возвращающая сила
Из второго закона Ньютона:
$$F = -m\omega_0^2 x = -kx$$
где $k = m\omega_0^2$ – **коэффициент возвращающей силы** (жёсткость).

### 3.2 Собственная частота и период
$$\omega_0 = \sqrt{\frac{k}{m}}, \quad T_0 = 2\pi\sqrt{\frac{m}{k}}$$

---

## **4. Энергия гармонического осциллятора**
Полная энергия постоянна:
$$E = K + U = \frac{mv^2}{2} + \frac{kx^2}{2}$$

После подстановки $x$ и $v$:
$$\large E = \frac{m\omega_0^2 A^2}{2}$$

**Энергия пропорциональна квадрату амплитуды и не зависит от времени.**

---

## **5. Примеры колебательных систем**
### 5.1 Груз на пружине
Уравнение: $\ddot{x} + \frac{k}{m}x = 0$  
Частота и период: $\omega_0 = \sqrt{\frac{k}{m}}$, $T_0 = 2\pi\sqrt{\frac{m}{k}}$

### 5.2 Физический маятник
Момент силы: $M = -mga\varphi$ (при малых углах)  
Уравнение: $J\ddot{\varphi} + mga\varphi = 0$  
Период: $T_0 = 2\pi\sqrt{\frac{J}{mga}}$

### 5.3 Математический маятник
Частный случай физического ($J = mL^2$, $a = L$):
$$\large T = 2\pi\sqrt{\frac{L}{g}}$$

### 5.4 Крутильные колебания
Момент: $M = -c\varphi$  
Уравнение: $J\ddot{\varphi} + c\varphi = 0$  
Период: $T_0 = 2\pi\sqrt{\frac{J}{c}}$

---

## **6. Затухающие колебания**
### 6.1 Сила сопротивления
При малых скоростях:
$$\vec{F}_{\text{сопр}} = -r\vec{v} = -r\dot{x}$$

### 6.2 Уравнение движения
$$\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = 0$$
где $\beta = \frac{r}{2m}$ – **коэффициент затухания**.

### 6.3 Решение (при слабом затухании: $\omega_0^2 > \beta^2$)
$$x(t) = A_0 e^{-\beta t} \sin(\omega t + \alpha)$$
где $\omega = \sqrt{\omega_0^2 - \beta^2}$ – частота затухающих колебаний.

### 6.4 Период затухающих колебаний
$$T = \frac{2\pi}{\omega} = \frac{2\pi}{\sqrt{\omega_0^2 - \beta^2}}$$
$T_{\text{затух}} > T_0$ – период увеличивается из-за сопротивления.

### 6.5 Логарифмический декремент затухания
$$\delta = \ln\frac{A(t)}{A(t+T)} = \beta T$$
– постоянная величина, характеризующая скорость затухания.

### 6.6 Режимы движения
1. **Слабое затухание** ($\omega_0^2 > \beta^2$): колебания с убывающей амплитудой
2. **Критическое затухание** ($\omega_0^2 = \beta^2$): возврат к равновесию без колебаний за минимальное время
3. **Сильное затухание** ($\omega_0^2 < \beta^2$): медленный апериодический возврат

---

## **7. Вынужденные колебания. Резонанс**
### 7.1 Уравнение вынужденных колебаний
$$\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = \frac{F_0}{m} \sin(\Omega t)$$
где $F_0\sin(\Omega t)$ – вынуждающая сила.

### 7.2 Общее решение
$$x(t) = \underbrace{A_0 e^{-\beta t} \sin(\omega t + \alpha)}_{\text{затухающие}} + \underbrace{A \sin(\Omega t + \Psi)}_{\text{вынужденные}}$$
Через некоторое время остаются только установившиеся вынужденные колебания.

### 7.3 Установившиеся колебания
$$\large x(t) = A \sin(\Omega t + \Psi)$$

**Амплитуда**:
$$A = \frac{F_0/m}{\sqrt{(\omega_0^2 - \Omega^2)^2 + 4\beta^2\Omega^2}}$$

**Сдвиг фаз**:
$$\tan\Psi = -\frac{2\beta\Omega}{\omega_0^2 - \Omega^2}$$

### 7.4 Резонанс
**Резонансная частота**:
$$\Omega_{\text{рез}} = \sqrt{\omega_0^2 - 2\beta^2}$$
При слабом затухании $\Omega_{\text{рез}} \approx \omega_0$.

**Резонансная амплитуда**:
$$A_{\text{рез}} = \frac{F_0}{2m\beta\sqrt{\omega_0^2 - \beta^2}} \approx \frac{F_0}{r\omega_0} \quad (\text{при } \beta \ll \omega_0)$$

**Свойства**:
- При $\beta \to 0$: $A_{\text{рез}} \to \infty$
- Чем меньше $\beta$, тем острее резонанс
- При резонансе $\Psi \approx -\pi/2$

---

## **8. Волны**
### 8.1 Основные определения
- **Волна** – распространение возмущений в пространстве
- **Упругая волна** – механическое возмущение в упругой среде

### 8.2 Типы волн
1. **Продольные**: колебания частиц вдоль направления распространения
   - Возникают в любых средах (твёрдых, жидких, газообразных)
   - Обусловлены деформациями сжатия/растяжения
   
2. **Поперечные**: колебания частиц перпендикулярно направлению распространения
   - Возникают только в твёрдых телах (и на поверхности жидкостей)
   - Обусловлены деформацией сдвига

### 8.3 Скорости волн
- **Поперечные** в твёрдых телах:  $$v_{\text{попер}} = \sqrt{\frac{G}{\rho}}$$
- **Продольные** в стержнях:  $$v_{\text{прод}} = \sqrt{\frac{E}{\rho}}$$

### 8.4 Уравнение плоской гармонической волны
Распространение вдоль $y$:
$$\large x(y,t) = A \sin\left[\omega\left(t - \frac{y}{v}\right)\right]$$

Альтернативные формы:
$$x = A \sin\left(\omega t - \frac{2\pi y}{\lambda}\right)$$
$$x = A \sin(\omega t - ky)$$

где:
- $v$ – фазовая скорость
- $\lambda$ – длина волны
- $k = \frac{2\pi}{\lambda}$ – волновое число

### 8.5 Связь параметров
$$\lambda = vT = \frac{v}{f}, \quad v = \lambda f = \frac{\omega}{k}$$

### 8.6 Дисперсия
Если $v$ зависит от частоты – среда обладает **дисперсией**.
Для звука в газах дисперсии нет.

---

## **9. Интерференция волн**
### 9.1 Когерентные волны
Волны когерентны, если:
1. Одинаковое направление колебаний
2. Одинаковая частота
3. Постоянная разность фаз

### 9.2 Разность хода и фаз
Для двух волн, приходящих в точку $B$:
$$\Delta = y_1 - y_2 \quad \text{(разность хода)}$$
$$\Delta\varphi = 2\pi\frac{\Delta}{\lambda} \quad \text{(разность фаз)}$$

### 9.3 Результирующая амплитуда
$$A = \sqrt{A_1^2 + A_2^2 + 2A_1A_2\cos(\Delta\varphi)}$$

### 9.4 Условия максимума и минимума
- **Максимум** (усиление):
  $$\Delta = n\lambda, \quad A = A_1 + A_2$$
- **Минимум** (ослабление):
  $$\Delta = (2n+1)\frac{\lambda}{2}, \quad A = |A_1 - A_2|$$
- **Полное гашение** (при $A_1 = A_2$):
  $$\Delta = (2n+1)\frac{\lambda}{2}, \quad A = 0$$

---

## **10. Итоговые формулы-шпаргалка**
### Колебания
$$\ddot{x} + \omega_0^2 x = 0 \quad \Rightarrow \quad x = A\sin(\omega_0 t + \alpha)$$
$$\omega_0 = \sqrt{\frac{k}{m}}, \quad T_0 = 2\pi\sqrt{\frac{m}{k}}$$
$$E = \frac{m\omega_0^2 A^2}{2}$$

### Затухающие колебания
$$\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = 0$$
$$x = A_0 e^{-\beta t} \sin(\omega t + \alpha), \quad \omega = \sqrt{\omega_0^2 - \beta^2}$$
$$\delta = \beta T$$

### Вынужденные колебания
$$\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = \frac{F_0}{m}\sin(\Omega t)$$
$$A = \frac{F_0/m}{\sqrt{(\omega_0^2 - \Omega^2)^2 + 4\beta^2\Omega^2}}$$
$$\tan\Psi = -\frac{2\beta\Omega}{\omega_0^2 - \Omega^2}$$

### Резонанс
$$\Omega_{\text{рез}} = \sqrt{\omega_0^2 - 2\beta^2}$$
$$A_{\text{рез}} = \frac{F_0}{2m\beta\sqrt{\omega_0^2 - \beta^2}}$$

### Волны
$$x(y,t) = A\sin(\omega t - ky)$$
$$\lambda = vT, \quad v = \lambda f, \quad k = \frac{2\pi}{\lambda}$$

### Интерференция
$$A = \sqrt{A_1^2 + A_2^2 + 2A_1A_2\cos\left(\frac{2\pi\Delta}{\lambda}\right)}$$
$$\Delta = n\lambda \text{ (макс)}, \quad \Delta = (2n+1)\frac{\lambda}{2} \text{ (мин)}$$

## **Ответы на вопросы раздела (кратко):**

1. **Гармоническое колебание**
	**Гармоническое колебание** — периодическое движение, при котором смещение изменяется по закону синуса/косинуса:
	$$x = A \sin(\omega_0 t + \alpha)$$
	**Характеризующие уравнения**:
	- Скорость: $v = A\omega_0 \cos(\omega_0 t + \alpha)$
	- Ускорение: $a = -A\omega_0^2 \sin(\omega_0 t + \alpha) = -\omega_0^2 x$
	- Диф. уравнение: $\ddot{x} + \omega_0^2 x = 0$

2. **Коэффициенты возвращающей силы и момента**
	- **Коэффициент возвращающей силы $k$**: $\vec{F} = -k\vec{x}$
	  *Физический смысл*: численно равен силе при смещении $x = 1$
	- **Коэффициент возвращающего момента $c$**: $\vec{M} = -c\vec{\varphi}$
	  *Физический смысл*: численно равен моменту при угловом смещении $\varphi = 1$

3. **Сохранение энергии**
	Полная энергия: $E = \frac{mv^2}{2} + \frac{kx^2}{2}$
	Подставляя $x = A\sin(\omega_0 t + \alpha)$ и $v = A\omega_0\cos(\omega_0 t + \alpha)$:
	$$E = \frac{m\omega_0^2 A^2}{2} = \text{const}$$
	→ энергия постоянна и пропорциональна квадрату амплитуды.

4. **Физический и математический маятники**
	- **Физический маятник**: твёрдое тело, колеблющееся вокруг точки ≠ центр масс
	  $$T_0 = 2\pi\sqrt{\frac{J}{mga}}$$
	- **Математический маятник**: материальная точка на невесомой нити
	  $$T = 2\pi\sqrt{\frac{L}{g}}$$

 5. **Диф. уравнение физического маятника**
	При малых углах $\varphi$:
	$$J\frac{d^2\varphi}{dt^2} = -mga\varphi$$
	$$\frac{d^2\varphi}{dt^2} + \frac{mga}{J}\varphi = 0$$

6. **Диф. уравнение крутильных колебаний**
$$J\ddot{\varphi} + c\varphi = 0$$
	где $c$ — коэффициент возвращающего момента.

 7. **Уравнения затухающих колебаний**
	- Диф. уравнение: $\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = 0$
	- Решение (слабое затухание): $x = A_0 e^{-\beta t} \sin(\omega t + \alpha)$
	где $\omega = \sqrt{\omega_0^2 - \beta^2}$

 8. **Коэффициент затухания**
$$\beta = \frac{r}{2m}$$
	где $r$ — коэффициент сопротивления среды.  
	**Зависит от**: сопротивления среды $r$ и массы $m$.

 9. **Логарифмический декремент затухания**
$$\delta = \ln\frac{A(t)}{A(t+T)} = \beta T$$
	— натуральный логарифм отношения двух последовательных амплитуд.

10. **Физический смысл $\beta$ и $\delta$**
	- $\beta^{-1}$ — время, за которое амплитуда уменьшается в $e$ раз
	- $\delta^{-1}$ — число колебаний, за которое амплитуда уменьшается в $e$ раз

 11. **Уравнения вынужденных колебаний**
	- Диф. уравнение: $\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = \frac{F_0}{m}\sin(\Omega t)$
	- Установившиеся колебания: $x = A\sin(\Omega t + \Psi)$

 12. **Частота установившихся вынужденных колебаний**
	Частота равна **частоте вынуждающей силы $\Omega$**, а не собственной частоте системы.

 13. **Резонанс**
	**Резонанс** — резкое увеличение амплитуды вынужденных колебаний при определённой частоте вынуждающей силы.  
	**Возникает**, когда $\Omega = \Omega_{\text{рез}} = \sqrt{\omega_0^2 - 2\beta^2}$

14. **Зависимость резонансной частоты**
$$\Omega_{\text{рез}} = \sqrt{\omega_0^2 - 2\beta^2}$$
	Зависит от:
	- Собственной частоты $\omega_0 = \sqrt{k/m}$
	- Коэффициента затухания $\beta = r/(2m)$

 15. **Упругая волна**
	**Упругая волна** — распространение механических возмущений в упругой среде.  
	**Скорости**:
	- Продольные: $v_{\text{прод}} = \sqrt{E/\rho}$ (зависит от модуля Юнга $E$ и плотности $\rho$)
	- Поперечные: $v_{\text{попер}} = \sqrt{G/\rho}$ (зависит от модуля сдвига $G$ и плотности $\rho$)

 16. **Уравнение плоской гармонической волны**
$$x(y,t) = A\sin(\omega t - ky)$$
	где:
	- $v = \omega/k$ — фазовая скорость
	- $\lambda = 2\pi/k$ — длина волны
	- $k$ — волновой вектор (модуль $k = 2\pi/\lambda$)

 17. **Когерентные волны**
	 Волны **когерентны**, если они:
		1. Имеют одинаковое направление колебаний
		2. Имеют одинаковую частоту
		3. Имеют постоянную разность фаз

18. **Разность хода при усилении (максимум)**
$$\Delta = y_1 - y_2 = n\lambda \quad (n = 0, 1, 2, ...)$$

 19. **Разность хода при ослаблении (минимум)**
$$\Delta = y_1 - y_2 = (2n + 1)\frac{\lambda}{2} \quad (n = 0, 1, 2, ...)$$

# **Ответы на вопросы раздела (подробно):**

## 1. Гармоническое колебание

**Гармоническое колебание** — это периодическое движение, при котором физическая величина (обычно смещение от положения равновесия) изменяется во времени по закону синуса или косинуса. Такое движение возникает в системах, где возвращающая сила пропорциональна смещению и направлена противоположно ему.

### Основное уравнение гармонического колебания:
$$x(t) = A \sin(\omega_0 t + \alpha)$$

где:
- $x(t)$ — смещение в момент времени $t$
- $A$ — **амплитуда колебаний** (максимальное абсолютное значение $x$)
- $\omega_0$ — **круговая (циклическая) частота** собственных колебаний
- $\alpha$ — **начальная фаза** (фаза в момент $t = 0$)
- $(\omega_0 t + \alpha)$ — **фаза колебаний**

### Характеризующие уравнения:

1. **Скорость гармонического колебания** (первая производная смещения):
$$v(t) = \frac{dx}{dt} = A\omega_0 \cos(\omega_0 t + \alpha)$$

2. **Ускорение гармонического колебания** (вторая производная смещения):
$$a(t) = \frac{d^2x}{dt^2} = -A\omega_0^2 \sin(\omega_0 t + \alpha)$$

3. **Связь между ускорением и смещением**:
$$a(t) = -\omega_0^2 x(t)$$
Это означает, что при гармонических колебаниях ускорение прямо пропорционально смещению и всегда направлено противоположно ему (к положению равновесия).

4. **Дифференциальное уравнение гармонических колебаний**:
$$\ddot{x} + \omega_0^2 x = 0$$
Функция $x = A \sin(\omega_0 t + \alpha)$ является решением этого уравнения.

---

## 2. Коэффициенты возвращающей силы и момента

### Коэффициент возвращающей силы $k$
**Определение**: В системах, совершающих гармонические колебания, возвращающая сила пропорциональна смещению:
$$\vec{F} = -k\vec{x}$$

**Физический смысл**: Коэффициент $k$ численно равен возвращающей силе, вызывающей смещение $x$, равное единице:
- Если $x = 1$, то $F = k$
- Размерность: Н/м (ньютон на метр)

**Пример**: Для пружины $k$ — это **жёсткость пружины**.

### Коэффициент возвращающего момента $c$
**Определение**: В системах, совершающих крутильные колебания, возвращающий момент пропорционален угловому смещению:
$$\vec{M} = -c\vec{\varphi}$$

**Физический смысл**: Коэффициент $c$ численно равен возвращающему моменту, вызывающему угловое смещение $\varphi$, равное единице:
- Если $\varphi = 1$ рад, то $M = c$
- Размерность: Н·м/рад

**Пример**: Для проволоки при крутильных колебаниях $c$ определяется её упругими свойствами: $c = \frac{\pi G r^4}{2L}$, где $G$ — модуль сдвига, $r$ — радиус, $L$ — длина.

## 3. Сохранение полной энергии при гармонических колебаниях

### Полная механическая энергия системы:
$$E = K + U = \frac{mv^2}{2} + \frac{kx^2}{2}$$
где:
- $K = \frac{mv^2}{2}$ — кинетическая энергия
- $U = \frac{kx^2}{2}$ — потенциальная энергия

### Подстановка гармонических функций:
1. $x = A \sin(\omega_0 t + \alpha)$
2. $v = A\omega_0 \cos(\omega_0 t + \alpha)$
3. $k = m\omega_0^2$

**Вычисление**:
$$E = \frac{m[A\omega_0 \cos(\omega_0 t + \alpha)]^2}{2} + \frac{m\omega_0^2 [A \sin(\omega_0 t + \alpha)]^2}{2}$$
$$E = \frac{mA^2\omega_0^2}{2}[\cos^2(\omega_0 t + \alpha) + \sin^2(\omega_0 t + \alpha)]$$

Используя тригонометрическое тождество $\cos^2\theta + \sin^2\theta = 1$:
$$\large E = \frac{m\omega_0^2 A^2}{2}$$

### Вывод:
- Полная энергия $E$ **не зависит от времени** (константа)
- Энергия пропорциональна **квадрату амплитуды** $A^2$
- Энергия пропорциональна **квадрату частоты** $\omega_0^2$
- Энергия перераспределяется между кинетической и потенциальной, но их сумма остаётся постоянной

---

## 4. Физический и математический маятники

### Физический маятник
**Определение**: Твёрдое тело произвольной формы, способное совершать колебания вокруг неподвижной горизонтальной оси, не проходящей через центр масс тела.

**Характеристики**:
- $m$ — масса маятника
- $J$ — момент инерции относительно оси вращения
- $a$ — расстояние от оси вращения до центра масс
- $\varphi$ — угловое смещение от положения равновесия

**Период малых колебаний**:
$$\large T_0 = 2\pi\sqrt{\frac{J}{mga}}$$

### Математический маятник
**Определение**: Частный случай физического маятника — материальная точка массой $m$, подвешенная на невесомой нерастяжимой нити длиной $L$.

**Характеристики**:
- Для математического маятника: $J = mL^2$ и $a = L$

**Период малых колебаний**:
$$\large T = 2\pi\sqrt{\frac{L}{g}}$$

**Важно**: Формула справедлива только при малых углах отклонения ($\varphi < 5^\circ - 10^\circ$), когда $\sin\varphi \approx \varphi$.

---

## 5. Дифференциальное уравнение колебаний физического маятника

### Вывод уравнения:

1. **Возвращающий момент силы тяжести**:
При отклонении маятника на угол $\varphi$ от вертикали, момент силы тяжести относительно точки подвеса:
$$M = -mga \sin\varphi$$
Знак минус означает, что момент направлен против отклонения (возвращающий момент).

2. **Упрощение для малых углов**:
Для малых углов ($\varphi < 10^\circ$): $\sin\varphi \approx \varphi$
$$M = -mga\varphi$$

3. **Основное уравнение динамики вращательного движения**:
$$M = J\epsilon = J\frac{d^2\varphi}{dt^2}$$
где $J$ — момент инерции, $\epsilon = \frac{d^2\varphi}{dt^2}$ — угловое ускорение.

4. **Подстановка**:
$$J\frac{d^2\varphi}{dt^2} = -mga\varphi$$

5. **Окончательное дифференциальное уравнение**:
$$\boxed{\frac{d^2\varphi}{dt^2} + \frac{mga}{J}\varphi = 0}$$

### Сравнение с уравнением гармонических колебаний:
Уравнение имеет вид $\ddot{\varphi} + \omega_0^2\varphi = 0$, где:
$$\omega_0 = \sqrt{\frac{mga}{J}} \quad \Rightarrow \quad T_0 = \frac{2\pi}{\omega_0} = 2\pi\sqrt{\frac{J}{mga}}$$

---

## 6. Дифференциальное уравнение гармонических крутильных колебаний

### Вывод уравнения:

1. **Возвращающий момент упругих сил**:
При закручивании проволоки на угол $\varphi$ возникает момент упругих сил:
$$M = -c\varphi$$
где $c$ — коэффициент возвращающего момента (зависит от материала и размеров проволоки).

2. **Основное уравнение динамики вращательного движения**:
$$M = J\epsilon = J\frac{d^2\varphi}{dt^2}$$

3. **Подстановка**:
$$J\frac{d^2\varphi}{dt^2} = -c\varphi$$

4. **Окончательное дифференциальное уравнение**:
$$\boxed{J\frac{d^2\varphi}{dt^2} + c\varphi = 0}$$
или
$$\boxed{\frac{d^2\varphi}{dt^2} + \frac{c}{J}\varphi = 0}$$

### Решение и параметры:
Решение: $\varphi = \varphi_0 \sin(\omega_0 t + \alpha)$
Круговая частота: $\omega_0 = \sqrt{\frac{c}{J}}$
Период: $T_0 = 2\pi\sqrt{\frac{J}{c}}$

---

## 7. Уравнения затухающих колебаний

### 1. Дифференциальное уравнение затухающих колебаний:
$$\boxed{\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = 0}$$
где:
- $\omega_0 = \sqrt{\frac{k}{m}}$ — собственная частота системы без затухания
- $\beta = \frac{r}{2m}$ — коэффициент затухания
- $r$ — коэффициент сопротивления среды

### 2. Решение для слабого затухания ($\omega_0^2 > \beta^2$):
$$x(t) = A_0 e^{-\beta t} \sin(\omega t + \alpha)$$
где:
- $A_0 e^{-\beta t}$ — амплитуда, экспоненциально убывающая со временем
- $\omega = \sqrt{\omega_0^2 - \beta^2}$ — частота затухающих колебаний
- $\alpha$ — начальная фаза

### 3. Период затухающих колебаний:
$$T = \frac{2\pi}{\omega} = \frac{2\pi}{\sqrt{\omega_0^2 - \beta^2}}$$

### 4. Важные замечания:
- Период затухающих колебаний $T > T_0$ (больше периода без затухания)
- С увеличением $\beta$ период увеличивается
- При $\beta \to \omega_0$ период стремится к бесконечности

---

## 8. Коэффициент затухания

### Определение:
$$\beta = \frac{r}{2m}$$
где:
- $r$ — коэффициент сопротивления среды (Н·с/м)
- $m$ — масса колеблющейся точки (кг)

### От чего зависит:
1. **Прямо пропорционален** коэффициенту сопротивления $r$:
   - Чем больше сопротивление среды, тем больше $\beta$
   - В более вязкой среде колебания быстрее затухают

2. **Обратно пропорционален** массе $m$:
   - При одинаковом сопротивлении более массивные тела дольше сохраняют колебания
   - Инерция массивного тела препятствует быстрому затуханию

3. **Не зависит** от амплитуды и частоты колебаний (для линейной силы сопротивления)

### Физическая интерпретация:
- $\beta$ характеризует скорость убывания амплитуды
- За время $\tau = \frac{1}{\beta}$ амплитуда уменьшается в $e \approx 2.718$ раз
- $\beta$ имеет размерность частоты (с⁻¹)

---

## 9. Логарифмический декремент затухания

### Определение:
Логарифмический декремент затухания $\delta$ — это натуральный логарифм отношения двух последовательных амплитуд, отстоящих друг от друга на период $T$:

$$\delta = \ln\frac{A(t)}{A(t+T)}$$

### Вычисление:
Для $A(t) = A_0 e^{-\beta t}$ и $A(t+T) = A_0 e^{-\beta(t+T)}$:

$$\delta = \ln\frac{A_0 e^{-\beta t}}{A_0 e^{-\beta(t+T)}} = \ln e^{\beta T} = \beta T$$

$$\boxed{\delta = \beta T}$$

### Свойства:
1. $\delta$ — безразмерная величина
2. $\delta$ постоянен для данной колебательной системы
3. $\delta$ не зависит от времени и номера колебания

---

## 10. Физический смысл коэффициента затухания и логарифмического декремента

### Коэффициент затухания $\beta$:
- **Физический смысл**: Величина, обратная времени $\tau$, за которое амплитуда уменьшается в $e$ раз:
  $$\tau = \frac{1}{\beta}$$
  За время $t = n\tau = \frac{n}{\beta}$ амплитуда уменьшается в $e^n$ раз.

- **Интерпретация**: $\beta$ характеризует **скорость затухания**:
  - Большое $\beta$ → быстрое затухание
  - Малое $\beta$ → медленное затухание

### Логарифмический декремент $\delta$:
- **Физический смысп**: Величина, обратная числу колебаний $N$, за которое амплитуда уменьшается в $e$ раз:
  $$N = \frac{1}{\delta}$$
  За $n$ колебаний амплитуда уменьшается в $e^{n\delta}$ раз.

- **Интерпретация**: $\delta$ показывает, **насколько сильно затухает система за одно колебание**:
  - Большое $\delta$ → сильное затухание за период
  - Малое $\delta$ → слабое затухание за период

### Связь между ними:
$$\delta = \beta T = \beta \cdot \frac{2\pi}{\omega} = \frac{2\pi\beta}{\sqrt{\omega_0^2 - \beta^2}}$$

---

## 11. Уравнения вынужденных колебаний

### 1. Дифференциальное уравнение вынужденных колебаний:
На систему действуют три силы:
- Упругая: $-kx$
- Сопротивления: $-r\dot{x}$
- Вынуждающая: $F_0 \sin(\Omega t)$

По второму закону Ньютона:
$$m\ddot{x} = -kx - r\dot{x} + F_0 \sin(\Omega t)$$

**Стандартная форма**:
$$\boxed{\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = \frac{F_0}{m} \sin(\Omega t)}$$
где:
- $\beta = \frac{r}{2m}$ — коэффициент затухания
- $\omega_0 = \sqrt{\frac{k}{m}}$ — собственная частота
- $\Omega$ — частота вынуждающей силы

### 2. Общее решение:
$$x(t) = \underbrace{A_0 e^{-\beta t} \sin(\omega t + \alpha)}_{\text{свободные затухающие}} + \underbrace{A \sin(\Omega t + \Psi)}_{\text{вынужденные установившиеся}}$$

### 3. Установившийся режим ($t \to \infty$):
Через достаточно большое время свободные колебания затухают, остаются только вынужденные:
$$\boxed{x(t) = A \sin(\Omega t + \Psi)}$$

где:
- **Амплитуда установившихся колебаний**:
  $$A = \frac{F_0/m}{\sqrt{(\omega_0^2 - \Omega^2)^2 + 4\beta^2\Omega^2}}$$

- **Сдвиг фаз** между колебаниями и вынуждающей силой:
  $$\tan\Psi = -\frac{2\beta\Omega}{\omega_0^2 - \Omega^2}$$

---

## 12. Частота установившихся вынужденных колебаний

### Важный вывод:
**Частота установившихся вынужденных колебаний всегда равна частоте вынуждающей силы $\Omega$, а не собственной частоте системы $\omega_0$.**

### Объяснение:
1. В установившемся режиме система "подстраивается" под внешнее воздействие
2. Вынуждающая сила совершает работу над системой, заставляя её колебаться со своей частотой
3. Собственные колебания системы затухают из-за сопротивления среды

### Математическое обоснование:
Из решения дифференциального уравнения:
$$x(t) = A \sin(\Omega t + \Psi)$$
Аргумент синуса — $\Omega t + \Psi$, что соответствует частоте $\Omega$.

### Практическое значение:
- Это позволяет управлять частотой колебаний системы, изменяя частоту вынуждающей силы
- Например, в электродвигателях частота вращения определяется частотой питающего тока

---

## 13. Резонанс

### Определение:
**Резонанс** — это явление резкого увеличения амплитуды вынужденных колебаний при приближении частоты вынуждающей силы $\Omega$ к определённому значению $\Omega_{\text{рез}}$, называемому резонансной частотой.

### Когда возникает:
Резонанс возникает при выполнении условия:
$$\Omega = \Omega_{\text{рез}} = \sqrt{\omega_0^2 - 2\beta^2}$$

### Механизм возникновения:
1. При $\Omega \approx \omega_0$ фаза вынуждающей силы такова, что она всегда "подталкивает" систему в направлении её движения
2. На каждом колебании вынуждающая сила совершает положительную работу
3. Энергия, поступающая от вынуждающей силы, максимальна при резонансе

### Условия для наблюдения резонанса:
1. Наличие вынуждающей силы
2. Слабое затухание ($\beta \ll \omega_0$)
3. Возможность изменения частоты вынуждающей силы

---

## 14. Зависимость резонансной частоты

### Резонансная частота:
$$\Omega_{\text{рез}} = \sqrt{\omega_0^2 - 2\beta^2}$$

### От чего зависит:

1. **От собственной частоты $\omega_0$**:
   $$\omega_0 = \sqrt{\frac{k}{m}}$$
   - Увеличивается с ростом жёсткости $k$
   - Уменьшается с ростом массы $m$
   - При $\beta = 0$: $\Omega_{\text{рез}} = \omega_0$

2. **От коэффициента затухания $\beta$**:
   $$\beta = \frac{r}{2m}$$
   - С увеличением $\beta$ резонансная частота уменьшается
   - При $\beta \geq \frac{\omega_0}{\sqrt{2}}$ резонанс исчезает
   - В реальных системах всегда $\beta > 0$, поэтому $\Omega_{\text{рез}} < \omega_0$

### Графическая зависимость:
- При малом $\beta$: $\Omega_{\text{рез}} \approx \omega_0$, резонанс острый
- При увеличении $\beta$: $\Omega_{\text{рез}}$ уменьшается, резонансная кривая расширяется и понижается
- При $\beta \to \frac{\omega_0}{\sqrt{2}}$: $\Omega_{\text{рез}} \to 0$, резонанс исчезает

---

## 15. Упругая волна

### Определение:
**Упругая волна** — это процесс распространения механических возмущений в упругой среде, при котором колебания частиц передаются от одной части среды к другой без переноса вещества в целом.

### Скорости упругих волн:

1. **Продольные волны** (колебания вдоль направления распространения):
   - В тонких стержнях: $$v_{\text{прод}} = \sqrt{\frac{E}{\rho}}$$
     где $E$ — модуль Юнга, $\rho$ — плотность среды
   - В газах: $$v = \sqrt{\frac{\gamma RT}{M}}$$
     где $\gamma$ — показатель адиабаты, $R$ — универсальная газовая постоянная, $T$ — температура, $M$ — молярная масса

2. **Поперечные волны** (колебания перпендикулярно направлению распространения):
   - В твёрдых телах: $$v_{\text{попер}} = \sqrt{\frac{G}{\rho}}$$
     где $G$ — модуль сдвига
   - На поверхности жидкости: зависит от глубины и ускорения свободного падения $g$

### От чего зависит скорость:
1. **От упругих свойств среды**:
   - Чем больше модуль упругости ($E$ или $G$), тем выше скорость
   - Твёрдые тела: наибольшая скорость
   - Жидкости: средняя скорость
   - Газы: наименьшая скорость

2. **От плотности среды $\rho$**:
   - Чем больше плотность, тем меньше скорость
   - При одинаковых упругих свойствах более лёгкие среды проводят волны быстрее

3. **От температуры** (для газов):
   - С ростом температуры скорость увеличивается

---

## 16. Уравнение плоской гармонической волны

### Основное уравнение:
Для волны, распространяющейся вдоль оси $y$ в положительном направлении:

$$\boxed{x(y,t) = A \sin\left[\omega\left(t - \frac{y}{v}\right)\right]}$$

### Альтернативные формы записи:

1. **Через длину волны $\lambda$**:
   $$x(y,t) = A \sin\left(\omega t - \frac{2\pi y}{\lambda}\right)$$

2. **Через волновой вектор $\vec{k}$**:
   $$x(y,t) = A \sin(\omega t - ky)$$

### Объяснение параметров:

1. **Фазовая скорость $v$**:
   - Скорость распространения постоянной фазы волны
   - Связь: $v = \frac{\omega}{k} = \lambda f$

2. **Длина волны $\lambda$**:
   - Расстояние между двумя ближайшими точками, колеблющимися в одинаковой фазе
   - Связь: $\lambda = vT = \frac{v}{f} = \frac{2\pi}{k}$

3. **Волновой вектор $\vec{k}$**:
   - Вектор, направленный вдоль распространения волны
   - Модуль: $k = \frac{2\pi}{\lambda}$ — волновое число
   - Смысл: $k$ показывает, сколько длин волн укладывается на $2\pi$ метров

### Волна, распространяющаяся в отрицательном направлении:
$$x(y,t) = A \sin\left[\omega\left(t + \frac{y}{v}\right)\right] = A \sin(\omega t + ky)$$

---

## 17. Когерентные источники и волны

### Определение:
Два источника колебаний называются **когерентными**, если они:
1. Излучают волны **одинаковой частоты**
2. Имеют **постоянную разность фаз** (не изменяющуюся со временем)
3. Колебания происходят в **одинаковом направлении**

### Волны, создаваемые когерентными источниками, также называются когерентными.

### Условия когерентности:

1. **Постоянство частот**:
   $$\omega_1 = \omega_2 = \omega$$

2. **Постоянство разности фаз**:
   $$\Delta\varphi = \varphi_2 - \varphi_1 = \text{const}$$

3. **Одинаковое направление колебаний**:
   Векторы колебаний параллельны или почти параллельны

### Примеры когерентных источников:
- Две щели, освещаемые одним источником света (опыт Юнга)
- Два громкоговорителя, подключённые к одному генератору
- Два маятника, запущенные синхронно

### Важность когерентности:
Только когерентные волны могут создавать **устойчивую интерференционную картину** с постоянным во времени распределением максимумов и минимумов.

---

## 18. Разность хода при усилении волн (интерференционный максимум)

### Условие максимума:
При интерференции двух когерентных волн **усиление** (максимум амплитуды) происходит в точках, где разность хода равна **целому числу длин волн**:

$$\boxed{\Delta = y_1 - y_2 = n\lambda \quad (n = 0, \pm 1, \pm 2, ...)}$$

где:
- $\Delta$ — разность хода (разность расстояний от источников до точки наблюдения)
- $\lambda$ — длина волны
- $n$ — порядок интерференционного максимума

### Физическое объяснение:
1. Разность хода $\Delta = n\lambda$ означает, что волны проходят пути, отличающиеся на целое число длин волн
2. Соответствующая разность фаз: $\Delta\varphi = 2\pi\frac{\Delta}{\lambda} = 2\pi n$
3. При $\Delta\varphi = 2\pi n$ волны приходят в одной фазе
4. Колебания складываются синфазно → амплитуда результирующего колебания максимальна

### Результирующая амплитуда в максимуме:
$$A_{\text{макс}} = A_1 + A_2$$

### Особый случай ($A_1 = A_2 = A_0$):
$$A_{\text{макс}} = 2A_0$$

---

## 19. Разность хода при ослаблении волн (интерференционный минимум)

### Условие минимума:
При интерференции двух когерентных волн **ослабление** (минимум амплитуды) происходит в точках, где разность хода равна **нечётному числу полуволн**:

$$\boxed{\Delta = y_1 - y_2 = (2n + 1)\frac{\lambda}{2} \quad (n = 0, \pm 1, \pm 2, ...)}$$

### Физическое объяснение:
1. Разность хода $\Delta = (2n+1)\frac{\lambda}{2}$ означает, что волны проходят пути, отличающиеся на нечётное число полуволн
2. Соответствующая разность фаз: $\Delta\varphi = 2\pi\frac{\Delta}{\lambda} = (2n+1)\pi$
3. При $\Delta\varphi = (2n+1)\pi$ волны приходят в противофазе
4. Колебания складываются в противофазе → амплитуда результирующего колебания минимальна

### Результирующая амплитуда в минимуме:
$$A_{\text{мин}} = |A_1 - A_2|$$

### Особый случай ($A_1 = A_2 = A_0$):
$$A_{\text{мин}} = 0 \quad \text{(полное гашение)}$$

### Графическое представление:
```
Максимумы:    Δ = 0, λ, 2λ, 3λ, ...
Минимумы:    Δ = λ/2, 3λ/2, 5λ/2, ...
```

Эта система условий лежит в основе всех интерференционных явлений и используется в интерферометрах, дифракционных решётках и других оптических приборах.
